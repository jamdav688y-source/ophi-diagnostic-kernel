# Audit Standard

## Objective

Ensure every consequential diagnosis can be reconstructed under scrutiny.

## Required Diagnostic Record

Each material diagnosis must record:

- system under review
- governing objective
- active constraints
- observed signals
- assumptions
- risk classification
- failure mode selected
- intervention pattern selected
- alternatives considered
- tradeoffs accepted
- escalation trigger status
- reversibility assessment

## Assumption Labels

All assumptions must be tagged as:
- Confirmed
- Probable
- Speculative

## Integrity Rules

A diagnostic artifact must not:
- conceal uncertainty
- inflate confidence
- ignore constraints
- omit alternatives when they materially exist
- pretend implementation authority where none exists

## Reconstruction Test

A third party should be able to answer:
- why this diagnosis was made
- what evidence supported it
- what constraints governed it
- what alternatives were rejected
- what intervention was recommended
